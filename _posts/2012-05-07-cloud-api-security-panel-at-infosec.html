This guest post comes from Andreas Krohn, API Specialist and consultant at Dopter.He blogs about APIs at Mashup.se (in Swedish) and develops tools to make APIs successful at APIHQ.Cloud API Security was the topic for a panel discussion at the Infosec conference in London April 26th.After a brief introduction of what APIs are, how companies are becoming platforms and what security implications this has the discussion mostly focused on how to secure mobile apps and how to keep security tokens protected.
