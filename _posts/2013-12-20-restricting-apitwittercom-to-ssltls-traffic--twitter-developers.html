This is an important notice for developers still using HTTP plaintext connections.On January 14th, 2014, connections to api.twitter.com will be restricted to TLS/SSL connections only.If your application still uses HTTP plaintext connections you will need to update it to use HTTPS connections, otherwise your app will stop functioning.
